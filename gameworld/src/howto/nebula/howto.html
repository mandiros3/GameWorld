1.Get The Emulator

 

·      Download Nebula Latest Version  http://nebula.emulatronia.com/descargas.php -- (V:2.25b)

·      Install Nebula  

 

2. Get The Roms

 

·      Download Roms at  http://www.romnation.net/srv/roms/mame103.html

 

3. Place the downloaded roms in the right place as well as bio

 

·      You should see a ROM folder inside your Nebula Folder after installing Nebula

·      Put all the downloaded roms in the ROM folder

 

o   IMPORTANT: Neogeo.zip & PGM.zip are bio zips and they must stay inside ROM Folder also,otherwise You Will Get Error

 

4. Configure the Controller

 

·      Load The Game First  

·      Goto Game > Configure Controls > Player 1

·      In Main Controls Box > Highlight The Key First > And Start Pressing Whatever You Want It to be on the controller > Save It.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

o   For future references, Make Sure To Save Your Input Configuration So You Can Load It On A Different Rom so that you don’t have to configure your controller on every single roms before you play.

Q: I can’t get my load my roms. What did I do wrong ?

A: Did You Install The Bio Yet ? If Not - Please Do So.

 

Q: Do We Need To Unzip The Rom After Download?

A: No, You Don't Need To, Just Place Them In The Rom Folder.

 

Q: How Do I Configure The Controller To Work With Nebula ?

A: First, You Need An Adapter And A PS2 Controller Or Any Other Type Of Controller is fine. Then, you just need to follow the step 4 in the detail instruction.
