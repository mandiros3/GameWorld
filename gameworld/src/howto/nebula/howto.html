1.Get The Emulator = The Console To Load Roms


* Download Nebula Latest Version = http://nebula.emulatronia.com/descargas.php -- (V:2.25b)
* Install It In Your C Drive > You Should Make The A Folder Call Nebula With All The Stuffs In It.



2. Get The Roms/games So You Can Play.


* Roms Download Link = http://www.romnation.net/srv/roms/mame103.html



3. In Side Your Nebula Folder


* You Should See A Roms Folder
* Put All The Roms In The Main ROMs Folder To Make Life Easier OR You Can Point The Directory Folder.
* IMPORTANT : Neogeo.zip + PGM.zip, Both 2 Are Bio Zip. They MUST Stay Inside The Roms Folder Also. Otherwise You Will Get Error



4. Must Configure The Controller Before You Can Play.. Lol


* Load The Game First > Game > Configure Controls > Player 1
* In Main Controls Box > Highlight The Key First > And Start Pressing Whatever You Want It To Be > Save It.
* For Future, Make Sure To Save Your Input Configuration So You Can Load It On A Different Rom.
